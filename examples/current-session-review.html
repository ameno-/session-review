Methodology

These rules are what make the review defensible.

Pre-audit first

Stop the main metrics at the audit request itself. Do not contaminate the review with the work required to generate it.

Activity beats wall-clock

Use contiguous execution intervals and strip long silent gaps. Session age is context, not the autonomy headline.

Coding vs execution

Separate validation and orchestration from real implementation time. Both matter, but they are not the same.

Conservative artifact counts

Prefer command-targeted outputs over noisy filesystem modification scans when the repo has unrelated churn.
